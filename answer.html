What are differences between Map, ForEach, Filter and Find

Map function used for loop through the array and make new array with matched items by transforming every element in an array individually. ForEach function used for loop through the array but it doesnot return any value. Filter Function filter creates a new array by removing elements that don't belong to the condition. Find function loop through an array and only return first match item.
